# 🚀 AI Interview Simulator (LLM Powered)

An advanced AI-powered interview preparation platform that simulates real HR + technical interviews using Large Language Models.

---

## 🌟 Features

- 📄 Resume-based question generation
- 🤖 AI-generated HR + Technical questions
- 🎤 Voice answer support (optional)
- ⏱️ Interview timer tracking
- 📊 Real-time scoring system
- 🧠 AI feedback & improved answers
- 🔁 Multi-question interview flow
- 📥 Downloadable report
- 🌐 Fully deployable (Frontend + Backend)

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- HTML, CSS
- Fetch API

### Backend
- FastAPI (Python)
- LLM Integration (Groq / OpenAI / Ollama)
- Resume Parsing (PDF/Text)
